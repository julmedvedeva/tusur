# Функциональная структура программы

Программа состоит из трёх файлов реализации: `timer.cpp` (класс `Time`),
`group.cpp` (класс `Group`), `main.cpp` (логика меню), и соответствующих
заголовочных файлов `timer.h`, `group.h`. Класс `Group` является контейнером
для объектов класса `Time` и управляет динамически выделенной памятью.

---

## Класс `Time` (`timer.h`, `timer.cpp`)

Представляет момент времени в пределах суток. Содержит три закрытых поля:
`hours`, `minutes`, `seconds`.

**Конструкторы:**
- `Time()` — конструктор по умолчанию, инициализирует время значением 00:00:00;
- `Time(int h, int m, int s)` — конструктор с параметрами;
- `Time(int totalSeconds)` — конструктор из общего числа секунд;
- `Time(const Time& other)` — конструктор копирования.

**Методы:**
- `setTime`, `setHours`, `setMinutes`, `setSeconds` — установка значений полей;
- `getHours`, `getMinutes`, `getSeconds` — получение значений полей;
- `toSeconds` — переводит время в общее количество секунд;
- `fromSeconds` — восстанавливает время из общего числа секунд;
- `deductSeconds` — вычитает заданное количество секунд из времени;
- `secondsBetween` — возвращает абсолютную разницу в секундах между двумя
  моментами времени;
- `toString` — возвращает строковое представление в формате ЧЧ:ММ:СС;
- `inputFromConsole` — запрашивает ввод с клавиатуры с проверкой допустимых
  диапазонов значений;
- `isValid` — статический метод проверки корректности значений часов, минут
  и секунд;
- `normalize` — закрытый вспомогательный метод, приводит значения полей к
  допустимому диапазону после любой арифметической операции, обеспечивая
  корректность представления времени (например, 61 секунда преобразуется в
  1 минуту 1 секунду).

**Перегруженные операторы:** `-` (вычитание секунд), `-=`, `=`, `==`, `<`,
`>`, `<<`, `>>`.

---

## Класс `Group` (`group.h`, `group.cpp`)

Представляет динамический массив объектов `Time`. Содержит закрытые поля:
`timers` (указатель на массив) и `size` (размер массива).

**Конструкторы:**
- `Group(int size)` — создаёт массив объектов `Time` заданного размера;
- `Group(const Group& other)` — конструктор копирования с глубоким
  копированием.

Конструктор копирования и оператор `=` реализуют глубокое копирование,
поскольку класс управляет динамической памятью (`new`/`delete[]`). Без них
копирование объекта `Group` привело бы к двойному освобождению памяти.

**Методы:**
- `setTimer` — устанавливает объект `Time` по индексу;
- `substractSeconds` — вычитает секунды из объекта по индексу;
- `getSize` — возвращает размер массива.

**Перегруженные операторы:** `[]` (доступ к элементу), `=` (присваивание
с глубоким копированием).

## Модуль `main.cpp`

Реализует консольное меню и логику взаимодействия с пользователем.

**Функции:**
- `showMenu` — выводит меню на экран;
- `subtractSecondsOperation` — реализует операцию вычитания секунд из времени;
- `calculateSecondsBetween` — реализует операцию подсчёта секунд между двумя
  моментами времени;
- `demonstrateAllOperations` — демонстрирует все конструкторы, операторы
  и класс `Group`;
- `main` — точка входа, содержит цикл `do-while` для повторного отображения
  меню до выбора пункта «Выход».

Глобальный флаг `g_verbose` управляет выводом диагностических сообщений
конструкторов и деструктора. В режиме демонстрации (пункт 3 меню) флаг
включается, что позволяет наблюдать жизненный цикл объектов `Time`.
