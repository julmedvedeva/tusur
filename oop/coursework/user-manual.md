# Руководство пользователя

## Запуск программы

После запуска исполняемого файла на экране отображается главное меню:

```
========================================
   ПРОГРАММА РАБОТЫ СО ВРЕМЕНЕМ
========================================
1. Вычитание секунд из времени
2. Подсчёт секунд между двумя моментами
3. Демонстрация всех операций
0. Выход
========================================
Выберите пункт меню:
```

Программа работает в цикле: после выполнения любой операции меню отображается
повторно. Для завершения работы необходимо выбрать пункт **0**.

---

## Пункт 1 — Вычитание секунд из времени

1. Введите исходное время тремя отдельными числами:
   - Часы: целое число от 0 до 23
   - Минуты: целое число от 0 до 59
   - Секунды: целое число от 0 до 59
2. Введите количество секунд для вычитания (неотрицательное целое число).

При вводе значения вне допустимого диапазона или нечислового символа программа
выводит сообщение об ошибке и повторно запрашивает ввод.

**Пример вывода результата:**
```
========================================
Результат:
Исходное время: 04:32:47
Вычитаемые секунды: 145
Оставшееся время: 04:30:22
========================================
```

---

## Пункт 2 — Подсчёт секунд между двумя моментами

1. Введите первый момент времени тремя отдельными числами (часы, минуты, секунды).
2. Введите второй момент времени тремя отдельными числами (часы, минуты, секунды).

Порядок ввода моментов не важен: программа возвращает абсолютную разницу.

**Пример вывода результата:**
```
========================================
Результат:
Первый момент времени: 12:10:12
Второй момент времени: 18:45:00
Разница в секундах: 23688
Что составляет: 6 ч. 34 мин. 48 сек.
========================================
```

---

## Пункт 3 — Демонстрация всех операций

Пользовательский ввод не требуется. Программа автоматически демонстрирует:

- работу всех конструкторов класса `Time`;
- преобразование времени в секунды;
- вычитание секунд и оператор `-=`;
- подсчёт разницы между двумя моментами;
- операторы сравнения (`<`, `>`, `==`);
- класс `Group` (массив объектов `Time`);
- форматирование вывода: ведущие нули в `toString` и операторе `<<`.

В этом режиме также отображаются сообщения о вызовах конструкторов и деструкторов,
что позволяет наблюдать жизненный цикл объектов.

---

## Пункт 0 — Выход

Программа выводит сообщение «Завершение программы...», затем предлагает нажать
Enter для закрытия окна консоли.
