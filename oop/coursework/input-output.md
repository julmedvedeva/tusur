# Описание структуры входных и выходных данных

## Входные данные

Программа принимает входные данные через консоль. Состав входных данных зависит от
выбранного пункта меню.

**Пункт 1 — Вычитание секунд из времени:**

- Исходное время задаётся тремя отдельными целыми числами:
  - Часы: целое число от 0 до 23
  - Минуты: целое число от 0 до 59
  - Секунды: целое число от 0 до 59
- Количество секунд для вычитания: неотрицательное целое число

**Пункт 2 — Подсчёт секунд между двумя моментами времени:**

- Два момента времени; каждый задаётся тремя отдельными целыми числами в тех же
  диапазонах: часы (0–23), минуты (0–59), секунды (0–59)

**Пункт 3 — Демонстрация всех операций:**

Пользовательский ввод не требуется. Программа использует заранее заданные значения.

Для всех числовых полей реализована проверка корректности ввода: при вводе значения
вне допустимого диапазона или нечислового символа программа выводит сообщение об
ошибке и повторно запрашивает ввод.

---

## Выходные данные

**Пункт 1 — Вычитание секунд из времени:**

- Исходное время в формате `ЧЧ:ММ:СС`
- Количество вычитаемых секунд
- Результирующее время в формате `ЧЧ:ММ:СС`

**Пункт 2 — Подсчёт секунд между двумя моментами времени:**

- Первый момент времени в формате `ЧЧ:ММ:СС`
- Второй момент времени в формате `ЧЧ:ММ:СС`
- Разница между моментами в секундах
- Та же разница, представленная в виде часов, минут и секунд

**Пункт 3 — Демонстрация всех операций:**

- Вывод значений объектов `Time`, созданных разными конструкторами
- Результаты операций: вычитание секунд, разница между моментами, сравнение,
  оператор `-=`
- Содержимое объекта `Group` (массив объектов `Time`)
- Сообщения о вызовах конструкторов и деструкторов, демонстрирующие жизненный цикл
  объектов
