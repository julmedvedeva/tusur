# Создание проекта в Visual Studio (Windows)

## 1. Создать проект

1. Открой Visual Studio
2. «Создать новый проект» → «Консольное приложение (C++)» → «Далее»
3. Имя проекта: `coursework`
4. Выбери расположение папки
5. Нажми «Создать»

## 2. Добавить файлы

Visual Studio создаст `coursework.cpp` по умолчанию — удали его.

Скопируй свои файлы в папку проекта (`coursework/coursework/`):
- `main.cpp`
- `timer.cpp`
- `timer.h`
- `group.cpp`
- `group.h`

Затем в Solution Explorer → правая кнопка на проекте → «Добавить» →
«Существующий элемент» → выбери все пять файлов.

## 3. Настроить кодировку (важно для русского текста)

Правая кнопка на проекте → «Свойства» → «C/C++» → «Командная строка» →
в поле «Дополнительные параметры» добавь:

```
/utf-8
```

## 4. Собрать

`Ctrl+Shift+B` или меню «Сборка» → «Собрать решение».

Готовый `.exe` будет в папке `Debug\` или `Release\`.
