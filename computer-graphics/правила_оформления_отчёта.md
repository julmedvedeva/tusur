# Правила оформления отчёта по лабораторной работе
## Дисциплина «Компьютерная графика», ТУСУР ФДО

---

## Структура отчёта

1. **Титульный лист** (по форме ниже)
2. **Оглавление**
3. **Введение** — формулировка задачи, цель, описание исходных данных (информация о графических объектах)
4. **Основная часть:**
   - описание используемой среды программирования
   - описание алгоритма решения задачи (пошаговое или в виде блок-схемы)
   - наиболее важные фрагменты листинга с комментариями
   - описание функциональных возможностей разработанного приложения
5. **Заключение** — краткие выводы о проделанной работе
6. **Приложения** — листинги программ и скриншоты рабочих окон
